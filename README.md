# 🚀 GBIT React

**GBIT React** é uma CLI profissional para criação de aplicações **React Enterprise**.

Com apenas um comando é possível gerar uma aplicação completa contendo:

- React
- Vite
- TypeScript
- Express
- PostgreSQL
- Prisma (Opcional)
- Docker (Opcional)
- JWT
- Swagger
- Upload de Arquivos
- Estrutura Enterprise

Tudo pronto para desenvolvimento.

---

# Instalação

```bash
npm install -g gbit-react
```

ou

```bash
npx gbit-react meu-projeto
```
